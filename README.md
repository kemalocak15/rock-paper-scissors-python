# Taş Kağıt Makas Oyunu (Python) 

Bu proje, Python öğrenme sürecimde geliştirdiğim, terminal üzerinden oynanabilen klasik bir Taş-Kağıt-Makas oyunudur. 

Kullanıcı bilgisayara karşı oynar ve oyun, kullanıcı çıkış yapmak isteyene kadar `while` döngüleri sayesinde kesintisiz devam eder. Hatalı tuşlamalara karşı giriş kontrolleri (input validation) eklenmiştir.

## Özellikler
* Sınırsız oyun döngüsü.
* Hatalı girişlerde programın çökmesini engelleyen hata kontrolleri.


## Kullanılan Teknolojiler
* Python 3.12.8
* `random` kütüphanesi (Bilgisayarın rastgele seçim yapması için)

##  Nasıl Çalıştırılır?
Bilgisayarınızda Python yüklüyse, terminal veya komut satırını açıp dosyanın bulunduğu dizine gidin ve aşağıdaki komutu çalıştırın:
`python tas_kagit_makas_oyunu.py`

## Öğrenim Çıktıları
Bu projeyi geliştirirken iç içe `while` döngülerini yönetmeyi, listelerdeki indeks (IndexError) hatalarını önlemeyi ve algoritma akışını mantıksal bloklara ayırmayı pratik ettim.
